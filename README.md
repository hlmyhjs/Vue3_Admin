## 前端运行

```bash
# 克隆项目
git clone https://github.com/qxnldh/Vue3_Admin.git

# 进入项目目录
cd vue3-admin

# 安装依赖
yarn

# 启动服务
yarn dev

# 构建测试环境 yarn build:stage
# 构建生产环境 yarn build:prod
# 前端访问地址 http://localhost:8080
```
